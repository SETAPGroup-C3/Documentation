
FlowPoint Documentation
=======================

Welcome to the official documentation for FlowPoint, a lightweight task management web application developed for SETAP.

Overview
--------

FlowPoint is a web-based application built using HTML, CSS, and JavaScript, with SQLite as the local backend. It was designed to help teams and individuals manage projects, tasks, and communications in one simple interface.

Key Features
------------

- Create and manage tasks and projects
- Track progress using task status labels (To Do, In Progress, Done)
- Chat functionality for direct and project-based communication
- Friend system with search and request handling
- SQLite-based persistent storage
- Designed for simplicity and cross-browser compatibility

Note: Although an AI-powered task assignment feature was initially proposed, it was not implemented in the final version.
